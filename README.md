# Smart Trash Can

React and Vite based dashboard project for a smart trash can.

## Features

* Visualize trash can fill levels
* Real-time sensor data display
* User-friendly dashboard interface
* Responsive design for desktop and mobile

## Technologies

* React 18
* Vite
* TailwindCSS (optional, for UI styling)
* Firebase / REST API (optional, for sensor data)

## Installation

```bash
# Clone the repository
git clone https://github.com/Busrwa/SmartBinWebsite.git
cd SmartBinWebsite

# Install dependencies
npm install

# Start development server
npm run dev
```

## Usage

Open your browser and go to `http://localhost:5173` to view the dashboard.
